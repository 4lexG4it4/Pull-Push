# Definizione di Push e Pull

## Push
Il termine **push** si riferisce a un metodo di distribuzione delle informazioni o dei dati, in cui il contenuto viene "spinto" attivamente verso gli utenti. In questo modello, Ã¨ l'emittente a decidere quando e come inviare le informazioni, senza che l'utente debba richiederle esplicitamente. Esempi comuni includono:

- Notifiche push su smartphone
- Email promozionali
- Feed RSS

## Pull
Il termine **pull** indica un metodo in cui gli utenti "tirano" attivamente le informazioni richieste. In questo caso, l'utente deve effettuare un'azione, come una ricerca o un clic, per ricevere il contenuto. Esempi comuni includono:

- Motori di ricerca
- Siti web e blog
- Download di file
